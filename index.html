The Augmented Narrative Toolkit

A development framework to develop data-driven highlighy interactive web documents.

It enables developers to add complex charts, maps and media elements and define the interaction between them using HTML, CSS and as little Javascript as possible.

I wanted to be able to add, for instance, a map (SVG, interactive) as if you were adding a simple image with an HTML tag like this: <img>
